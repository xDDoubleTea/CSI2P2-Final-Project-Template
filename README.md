## Resource

- Allegro download: [https ://github.com/liballeg/allegro5/releases](https://github.com/liballeg/allegro5/releases)
- Allegro install(Mac OS): [https://hackmd.io/@Jiza/BkZ5a5yL2](https://hackmd.io/@Jiza/BkZ5a5yL2)
- Allegro documentation: [https://liballeg.org/a5docs/trunk/](https://liballeg.org/a5docs/trunk/)
- GIF convert: [https://ezgif.com/video-to-gif](https://ezgif.com/video-to-gif)

> The linux build was only tested on ubuntu and it uses apt as the package manager. I have added an install script for arch linux and it uses yay to install allegro.
> There is no install script provided for other distributions so good luck if you are using fedora or gentoo :)
